# E-commerce Data Engineering Pipeline

## Overview
This project implements an end-to-end data engineering pipeline for an e-commerce analytics platform.

## Architecture
- Raw data generation using Python
- Staging, production, and warehouse schemas in PostgreSQL
- Star schema with fact and dimension tables
- Power BI dashboards for analytics

## Tech Stack
- Python
- PostgreSQL
- Power BI
- GitHub Actions
- Docker

## How to Run
1. Generate data
2. Load data into PostgreSQL
3. Transform and validate data
4. Query warehouse
5. Visualize in Power BI
